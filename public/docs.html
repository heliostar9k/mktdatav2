Market Data API Documentation

Search Endpoint

POST https://mktdata-production.up.railway.app/api/search

Request Format

{
  "query": "your search string here"
}

Example Queries

  • "show me live nasdaq patterns"
  • "any insights about AMGN"
  • "generate a trade signal"
  • "what are the strongest patterns"

Example Usage

cURL

curl -X POST https://mktdata-production.up.railway.app/api/search \ -H "Content-Type: application/json" \ -d '{"query": "show me live nasdaq patterns"}'

Python

import requests

response = requests.post(
    'https://mktdata-production.up.railway.app/api/search',
    json={'query': 'show me live nasdaq patterns'}
)
data = response.json()

JavaScript

fetch('https://mktdata-production.up.railway.app/api/search', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json'
    },
    body: JSON.stringify({
        query: 'show me live nasdaq patterns'
    })
})
.then(response => response.json())
.then(data => console.log(data));

Response Format

{
    "results": [
        {
            "ticker": "string",
            "company_name": "string",
            "live_status": "Yes/No",
            "pattern_keywords": "string",
            "pattern_description": "string",
            "pattern_strength": number
        }
    ],
    "analysis": {
        "understanding": {
            "query_type": "string",
            "time_sensitive": boolean,
            "requires_live_data": boolean
        },
        "search_strategy": object,
        "result_count": number
    },
    "additional_insights": "string (when applicable)"
}

Response Codes

  • 200 - Success
  • 400 - Bad request (missing query)
  • 500 - Internal server error
