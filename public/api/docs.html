API Documentation

Authentication

All API requests require an API key to be included in the header:

{
    "x-api-key": "your_api_key"
}

Get your API key from the dashboard after creating an account.

Search Endpoint

POST /api/search

Search for market patterns and insights using natural language queries.

Request Format

{
    "query": "your search query here"
}

Response Format

{
    "results": [
        {
            "ticker": "string",
            "company_name": "string",
            "live_status": "Yes/No",
            "pattern_keywords": "string",
            "pattern_description": "string",
            "pattern_strength": number
        }
    ],
    "analysis": {
        "understanding": {
            "query_type": "string",
            "time_sensitive": boolean,
            "requires_live_data": boolean
        },
        "result_count": number
    },
    "additional_insights": "string (when applicable)"
}

Query Types

Pattern Search

Search for specific market patterns

"show me live nasdaq patterns"

Stock Analysis

Get insights about specific stocks

"tell me about AMGN patterns"

Signal Detection

Find active trading signals

"show me strong bullish signals"

Example Usage

cURL

curl -X POST https://mktdatav2-production.up.railway.app/api/search \
-H "Content-Type: application/json" \
-H "x-api-key: your_api_key" \
-d '{
    "query": "show me live nasdaq patterns"
}'

Python

import requests

response = requests.post(
    'https://mktdatav2-production.up.railway.app/api/search',
    headers={
        'Content-Type': 'application/json',
        'x-api-key': 'your_api_key'
    },
    json={
        'query': 'show me live nasdaq patterns'
    }
)

data = response.json()

JavaScript

fetch('https://mktdatav2-production.up.railway.app/api/search', {
    method: 'POST',
    headers: {
        'Content-Type': 'application/json',
        'x-api-key': 'your_api_key'
    },
    body: JSON.stringify({
        query: 'show me live nasdaq patterns'
    })
})
.then(response => response.json())
.then(data => console.log(data));

Response Codes

  • 200 Successful request
  • 400 Bad request (missing or invalid parameters)
  • 401 Unauthorized (missing or invalid API key)
  • 429 Too many requests
  • 500 Internal server error
